# Template: Blog Post SEO

**Departamento:** CONTENT
**Uso:** Crear artículos optimizados para SEO y conversiones

---

## Input Requerido

```yaml
tema: "Tema del artículo"
audiencia: "Descripción de la audiencia objetivo"
objetivo: "awareness|engagement|conversion|retention"
keywords:
  - "keyword principal"
  - "keyword secundaria 1"
  - "keyword secundaria 2"
tono: "profesional|casual|tecnico|inspiracional"
longitud: "1500|2000|2500|3000" (palabras)
```

---

## Estructura de Output

```markdown
# [H1 con Keyword Principal]

[Introducción hook - 100-150 palabras]
- Dato impactante O pregunta provocadora
- Problema que resuena con audiencia
- Promesa de lo que aprenderán

## [H2: Problema/Contexto]

[Desarrollo del problema con ejemplos]
- Estadísticas relevantes
- Historia/ anecdota
- Por qué importa ahora

## [H2: Solución/Framework]

[Contenido principal - 40% del artículo]
- Framework numerado o pasos
- Ejemplos concretos
- Tips prácticos

## [H2: Evidencia/Casos]

[Validación de la solución]
- Case study o ejemplo real
- Datos que soportan
- Testimonios (si disponible)

## [H2: Implementación Práctica]

[Acciones concretas]
- Checklist de pasos
- Herramientas recomendadas
- Errores comunes a evitar

## [H2: Conclusión]

[Resumen + CTA]
- 3 puntos clave
- Próximos pasos
- CTA claro (comentar, compartir, contactar)

---
## Meta Datos

**Meta title:** [60 caracteres máximo, incluir keyword principal]
**Meta description:** [155 caracteres, incluir keyword + CTA]
**Keywords usadas:** [lista]
**Readability score:** [Flesch 60-70]
**Palabras:** [conteo]
**CTAs incluidos:** [número y ubicación]
```

---

## Ejemplo de Uso

**Input:**
```yaml
tema: "Beneficios de implantes dentales"
audiencia: "Adultos 40-60 años considerando tratamiento dental"
objetivo: "conversion"
keywords:
  - implantes dentales
  - salud bucal
  - sonrisa perfecta
tono: "profesional pero accesible"
longitud: "2000"
```

**Output esperado:**
- Artículo 2,000 palabras
- Estructura con H2 claros
- 3 CTAs distribuidos
- Meta title/description optimizados
- Keywords integradas naturalmente
