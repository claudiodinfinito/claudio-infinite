## 📊 LECCIONES CRÍTICAS

### 1. Skills vs Exec
**Skill específico > exec genérico SIEMPRE**
- Error: spawnear subagentes para investigación
- Correcto: usar Skill Gemini
- Error: usar exec gh directo
- Correcto: usar Skill GitHub

### 2. GLM-5 Limitaciones
**GLM-5 = conversación/redacción, NO ejecución frágil**
- ✅ Bien en: español, síntesis, ideación
- ❌ Mal en: tools estrictas, tareas frágiles, consistencia operativa
- Solución: barreras (formatos fijos, pasos cortos, toolset limitado)

### 3. Arquitectura Híbrida
**Skills (plantillas) + Subagentes (workers)**
- Skills: pocas pero efectivas, inputs/outputs claros
- Subagentes: misión única + toolset mínimo
- Main (GLM-5): router, no ejecutor
