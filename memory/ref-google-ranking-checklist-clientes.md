# Google Ranking Checklist para Clientes

**Fuente:** Google Leak 2,590 archivos (2024)
**Propósito:** Lista práctica de factores de ranking verificables

---

## 🔥 Alta Prioridad (Impacto Directo)

### 1. Anchor System (CRÍTICO)
- [ ] Anchors contienen keyword objetivo
- [ ] Diversidad de anchors (no todos iguales)
- [ ] Evitar anchors spammy
- [ ] Revisar anchors de competencia

### 2. Content Quality
- [ ] Longitud adecuada según intención
- [ ] E-E-A-T signals (experiencia, autoridad)
- [ ] Freshness (actualizaciones recientes)
- [ ] Sin thin content

### 3. Technical Foundation
- [ ] Core Web Vitals (LCP, FID, CLS)
- [ ] Mobile-first indexing
- [ ] HTTPS obligatorio
- [ ] Schema markup implementado

---

## 📊 Media Prioridad (Señales de Autoridad)

### 4. Link Profile
- [ ] Calidad > Cantidad
- [ ] Relevancia temática
- [ ] Diversidad de dominios
- [ ] Audit regular con Ahrefs/Semrush

### 5. User Signals
- [ ] CTR en SERPs
- [ ] Tiempo en página
- [ ] Bounce rate razonable
- [ ] Búsquedas de marca

### 6. Entity Optimization
- [ ] Google Business Profile completo
- [ ] NAP consistente (Name, Address, Phone)
- [ ] Citaciones en directorios
- [ ] Reviews recientes

---

## 🧪 Baja Prioridad (Nice to Have)

### 7. Structured Data
- [ ] FAQ schema
- [ ] Review schema
- [ ] LocalBusiness schema
- [ ] Article schema

### 8. Content Velocity
- [ ] Publicación regular
- [ ] Actualización contenido viejo
- [ ] Topic clusters
- [ ] Internal linking

---

## ⚠️ Lo que NO importa (según leak)

| Factor | Veredicto |
|--------|-----------|
| Meta keywords | ❌ Ignorado |
| Exact match domains | ⚠️ Mínimo impacto |
| Keyword density | ❌ No medir |
| PageRank sculpting | ❌ Obsoleto |

---

## 📋 Auditoría Rápida (30 min)

1. **Screaming Frog** → Technical issues
2. **PageSpeed Insights** → Core Web Vitals
3. **Search Console** → Indexación, errores
4. **Ahrefs** → Backlinks, anchors
5. **Manual review** → Content quality

---

## 🎯 Acción Post-Auditoría

1. Fix critical errors first
2. Optimize top 10 páginas
3. Crear plan de contenido
4. Monitor mensual
