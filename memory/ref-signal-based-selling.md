# Signal-Based Selling Reference

**Propósito:** Metodología de ventas basada en señales de comportamiento.

## ¿Qué es Signal-Based Selling?

Ventas basadas en **señales de intención** que indican cuándo un prospecto está listo para comprar.

| Ventas Tradicionales | Signal-Based Selling |
|---------------------|---------------------|
| Llamadas en frío | Responder a señales |
| Mismo mensaje para todos | Mensaje según comportamiento |
| Intuición | Datos |
| Alto rechazo | Alta conversión |

---

## Tipos de Señales

| Señal | Ejemplo | Acción |
|-------|---------|--------|
| **Behavioral** | Visitó página pricing | Email con oferta |
| **Intent** | Buscó "alternativa a X" | Outreach proactivo |
| **Firmographic** | Empresa creció 50% | Upsell/cross-sell |
| **Engagement** | Abrió 3 emails | Call de ventas |
| **Timing** | Fin de Q4 | Cierre rápido |

---

## HubSpot Signals

HubSpot rastrea automáticamente:
- Visitas a páginas clave
- Cambios en la empresa (funding, hiring)
- Engagement con contenido
- Buying intent data

**Uso:** Configurar workflows que se disparan con estas señales.

---

## ClickFunnels Approach

ClickFunnels usa funnels que:
- Guían al prospecto psicológicamente
- Crean "triggers" basados en comportamiento
- Segmentan por acciones tomadas

**Tipos de funnels:**
- Tripwire (oferta baja → upsell)
- Webinar (registro → venta)
- Lead magnet (gratis → secuencia)

## Implementación para Clientes

### Paso 1: Identificar Señales Clave

| Cliente | Señales que importan |
|---------|---------------------|
| Clínica dental | Visitó página tratamientos, descargó guía |
| SaaS | Trial expirando, usó feature premium |
| Ecommerce | Abandonó carrito, vio producto 3x |

### Paso 2: Configurar Triggers

```
Señal detectada → Workflow automático → Acción
```

### Paso 3: Medir Conversión

- Lead → MQL conversion rate
- MQL → SQL conversion rate
- Time to close
