# Claude Conversations Archive

**Propósito:** Almacenar conversaciones con Claude que Daniel quiera compartir conmigo.

---

## Cómo usar

1. Pega el texto de la conversación en un archivo aquí
2. Nombra el archivo: `YYYY-MM-DD__tema.md`
3. Yo lo leeré y extraeré lo relevante

---

## Formato sugerido

```markdown
# Conversación con Claude - YYYY-MM-DD

**Tema:** [descripción breve]

**Fecha:** [fecha]

**Contexto:** [por qué es relevante]

---

[pegar conversación aquí]

---
```
